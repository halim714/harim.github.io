# CLAUDE.md — Meki 에이전트 핵심 규칙

> 이 파일은 Claude Code 세션 시작 시 자동 로드됩니다. 최소한의 필수 규칙만 담고 있습니다.
> 세부 지침은 `.agents/skills/`와 `.agents/roles/`를 참조하세요.

---
## 필수 시작 프로토콜

**작업 전 반드시 읽을 것:**
1. **`PLAN.md`** — 전체 계획, 태스크별 담당 Role과 상태
2. **`PROGRESS.md`** — 현재 진행 상황, 완료 목록, 실패 이력

---
## 절대 금지 사항

1. **사용자에게 질문하지 마라** — 백그라운드 자동화 세션이므로 질문 시 무한 대기
2. **종료되지 않는 프로세스 실행 금지** — `npm run dev` ❌, `npm run build` ✅
3. **기존 코드를 삭제하지 마라** — 수정 시 `.agents/skills/safe-edit/SKILL.md` 참조
4. **스코프 밖 파일 수정 금지** — Role별 스코프는 `.agents/skills/architecture-context/SKILL.md` 참조

---
## 태스크 완료 규칙

1. 할당된 태스크**만** 수행하라
2. `npm run build`로 빌드 확인
3. `PROGRESS.md`에 완료 기록 추가
4. 결과 요약 출력 후 **즉시 종료** — 추가 질문/제안 금지
