# start-session

세션 시작 시 반드시 실행하는 초기화 커맨드입니다.
PLAN.md와 PROGRESS.md를 읽어 현재 맡아야 할 태스크를 파악합니다.

**실행 절차:**

1. 다음 두 파일을 순서대로 읽어라:
   - `/Users/halim/Desktop/meeki/meki/PLAN.md`
   - `/Users/halim/Desktop/meeki/meki/PROGRESS.md`

2. 읽은 후 다음 형식으로 현황을 요약해라:

```
## 세션 초기화 완료

**현재 Phase**: [PLAN.md에서 확인한 현재 Phase]
**내 담당 태스크**: [PROGRESS.md 오케스트레이터 메모 또는 PLAN.md에서 확인]
**태스크 ID**: [예: P1-T1]
**의존 태스크 완료 여부**: [확인 결과]
**시작 가능 여부**: ✅ 즉시 시작 / ⚠️ 대기 필요 (이유: ___)
```

3. 시작 가능한 경우 작업을 즉시 시작하라. 대기가 필요한 경우 이유를 명확히 보고하라.
